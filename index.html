Gong enforces 3 API calls/sec and 10,000 API calls/day per key. High-frequency syncs on large orgs risk hitting this ceiling and causing failures.
Fivetran re-imports the TRANSCRIPT table on a monthly basis. Speaker ID updates, custom dictionary edits, or corrected transcripts may take up to a month to appear in your warehouse.
Fivetran added CALL_OUTLINE, CALL_OUTLINE_ITEM, CALL_HIGHLIGHT, CALL_HIGHLIGHT_ITEM, and CALL_KEY_POINT in June 2025. These tables are not in the Gong Data Cloud schema — Fivetran is the only way to get them.
All Fivetran connections sharing one API key also share the 10,000 call/day quota. Separate keys give each connection its own independent limit.
Fivetran has introduced multiple breaking schema changes to the Gong connector — tables merged, renamed, and split across versions. Hardcoded queries will break without a monitoring strategy.
